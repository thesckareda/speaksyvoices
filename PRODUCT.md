# Speaksy Voices

## What
Premium web app that converts conversational Markdown into realistic AI phone-call audio.

## Who
Product teams, CX designers, and voice QA teams who need multi-language demo calls from scripted scenarios.

## Platform
web

## Register
product

## Stack
- Next.js (App Router) + TypeScript + Tailwind CSS
- DeepSeek V4 Flash for multi-conversation Markdown analysis
- Cartesia Sonic TTS / voice cloning for Agent + User speech
- SSE for live generation progress

## Brand
- Mood: midnight studio booth — quiet glass, indigo signal light
- Seed: oklch violet/indigo (~270°)
- Strategy: restrained · pure surfaces · brand in primary + accent
